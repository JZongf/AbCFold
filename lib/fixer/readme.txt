g++:
  g++ Fixer2.cpp -O3 -o Fixer2 -Wno-deprecated

usage:
  Fixer2 -i [Input PDB]  -o [Output PDB] [option] 

  [option]:
    -s   input fas file.
         If there are multiple chains, combine them into one chain according to the order.
    -t   template file.
    -h   help info.

  Note: Fixer2为非交互的命令程序

例子1：
>./Fixer2 -i ./test/test_2.pdb -o ./test/test_2_fix_notpl.pdb                     
Success: Repair main chain atom: 2 VAL O

Warning: there are many missing amino acids, it is recommended to enter the template file.
